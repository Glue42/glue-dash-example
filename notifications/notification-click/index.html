Notification Click

The applications below demonstrates handling an interop method invocation when a notification is clicked.

On load, Application B registers an interop method called "HandleNotificationClick". Use the input field in Application A to enter a message that will be set as "HandleNotificationClick" method's argument. Click "Raise Notification" button. When the notification appears, click on it. The message entered in the input field will be displayed in Application B.
