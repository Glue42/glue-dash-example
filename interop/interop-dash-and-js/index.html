Basic Interoperability between Dash and JavaScript

The applications demonstrate interoperability between a Dash and JavaScript applications. A Dash application invokes an interop method registered in a JavaScript application.

On load, Application B (the JavaScript application) registers a method called "SendMessage" which accepts one parameter "message". Click the "Send" button in Application A (the Dash application) to invoke this method. The value in the input field labeled "Enter a message" will be printed in Application B.
